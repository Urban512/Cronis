# Cronis Design Language

## Philosophy

- Premium
- Minimal
- Apple-inspired
- Fast
- Clean
- Consistent
- Functional

## Principles

Every component must:

- Use ThemeManager colors
- Respect spacing constants
- Support dark themes
- Be reusable
- Never hardcode colors
- Never hardcode spacing
- Never hardcode fonts

## Motion

Animations must be:

100ms
150ms
200ms
250ms

No animation should feel distracting.

## Corners

Cards:
16px

Buttons:
12px

Search:
12px

Sidebar:
16px

Window:
20px

## Shadows

Very soft.

Never strong.

## Blur

Background only.

Never blur components.
