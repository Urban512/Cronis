# Cronis Design System

## Philosophy

Modern.
Minimal.
Fast.
Readable.

Every element must feel lightweight and polished.

Animations should be subtle.

Never overwhelm the player.

Everything should look like one coherent application.

---

## Design Goals

- Rounded corners
- Soft shadows
- Blur background
- Dark theme first
- Accent color support
- Theme system
- Smooth animations
- Keyboard navigation
- Search everywhere
