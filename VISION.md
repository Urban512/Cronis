# Cronis Vision

## Mission

Create the cleanest, most modern and most customizable HUD experience for Hypixel SkyBlock.

Cronis is designed from the ground up with a focus on premium design, performance and consistency.

Instead of adding hundreds of random features, every feature should improve the overall player experience.

---

# Vision

Cronis should become the go-to HUD framework for modern Hypixel SkyBlock.

The project prioritizes:

- Beautiful design
- Excellent performance
- Thoughtful user experience
- Long-term maintainability
- Community driven themes
- Modular architecture

---

# Core Philosophy

> Clean. Modern. Customizable.

---

# Design Principles

Every pixel must have a purpose.

If it doesn't look premium, it doesn't ship.

Great by default. Powerful when needed.

Performance is a feature.

Consistency over complexity.

---

# Project Goals

• Always support the latest Hypixel SkyBlock version.

• Keep the project 100% client-side.

• Never automate gameplay.

• Never violate Hypixel rules.

• Deliver a polished user experience.

---

# Long Term Goals

- Premium HUD Framework

- Theme Engine

- Layout Profiles

- Community Themes

- Beautiful Animations

- Apple-inspired UI

- Complete Hypixel SkyBlock support

---

# What Cronis IS

Cronis is a modern HUD framework built exclusively for Hypixel SkyBlock.

---

# What Cronis IS NOT

Cronis is not a cheat.

Cronis is not a macro.

Cronis does not automate gameplay.

Cronis does not modify packets.

Cronis respects Hypixel's rules.
